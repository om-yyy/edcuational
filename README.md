# 🎓 Educational App Core for Android 📱✨

Welcome to the **Educational App Core for Android**! 🚀 This is an open-source, beautifully look-maxxed educational framework engineered entirely with **Jetpack Compose** 🪐. It delivers dynamic course discovery and beautiful video interfaces wrapped in an immersive dark cyberpunk ecosystem! 🌌💜

---

### 🌟 Key Features 💎

- 🌌 **Premium Dark UI Theme:** Immersive deep amethyst night-blue gradients tailored for developers and students who code/learn till late night! 🌙✨
- 🔮 **Glassmorphic Components:** High-end modern cards wrapped in custom, ultra-thin glowing neon borders! 🎨🧪
- 🧠 **Smart Multiselection:** Fully responsive selection logic inside `InterestScreen` to efficiently pass user tastes! 🎯📥
- ⚡ **Performance Guarded:** Armed with robust `remember` snapshot state blocks to block heavy loops and lag on recomposition! 🛡️⚡
- 📚 **Curated Course Streams:** Bundled with up to 30 searchable, real YouTube video lectures from famous tech channels! 🎥🔥

---

### 📱 Get The App 📥

* 📦 **Download the APK:** Hop directly onto the 🎉 [Releases](https://github.com/om-yyy/edcuational/releases) section and install the latest compiled build!
* 🛠️ **Installation Note:** Simply ignore the standard Android Play Protect warning by hitting *Install Anyway*! 🔓📱





---

### 📘 Architecture Breakdown ⚙️

- 🗄️ **Data Layer:** Fast & clean singletons stored under `DummyData.kt` 📂
- 🧱 **UI Ecosystem:** Clean separation across `InterestScreen.kt` 🎯 -> `CourseScreen.kt` 📚 -> `VideoScreen.kt` 🎬
